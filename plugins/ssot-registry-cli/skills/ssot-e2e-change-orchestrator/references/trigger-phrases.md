# Trigger Phrases

This skill should trigger when the request includes cross-phase language such as:

- "ADR + SPEC + features + boundary + freeze"
- "schema change + migration + tests + link everything"
- "decision to release"
- "decision to closure"
- "certifiably fully featured"
- "certifiably fully conformant"
- "create the docs, the features, the boundary, freeze it, implement it, test it, and release it"
