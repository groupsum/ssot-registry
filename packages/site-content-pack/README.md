# @ssot-registry/site-content-pack

Private content package for the SSOT Registry website.

The package owns the product identity, pages, navigation, proof content, FAQ, and structured-data intent declarations consumed by the website host. The host renders this package through published `@mdwrk/lander-*` packages.

Run `npm run plan:xlsx` to regenerate `artifacts/ssot-registry-content-plan.xlsx`, the combinatorial AEO/SEO/AiEO content plan for the generated page corpus.
