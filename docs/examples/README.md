# Examples

- `minimal-repo` shows a minimal empty but valid registry.
- `advanced-repo` shows a non-trivial feature, claim, test, evidence, boundary, and release graph.
- `e2e-three-releases.md` shows an init-to-`rel:3.0.0` CLI rail with strict release sequencing.
- `formats-and-exports.md` shows CLI rendering options and export workflows for lists, full registry, and graph artifacts.
