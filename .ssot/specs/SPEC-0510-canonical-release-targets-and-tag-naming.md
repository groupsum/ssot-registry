# SPEC-0510: Canonical release targets and tag naming

## Status
Draft

## Canonical targets

- `pkgs/ssot-contracts`
- `pkgs/ssot-views`
- `pkgs/ssot-codegen`
- `pkgs/ssot-registry`
- `pkgs/ssot-cli`
- `pkgs/ssot-tui`

`pkgs/ssot-registry` is the canonical Python runtime release target.

## Tag format

Each package SHALL publish with a package-specific tag:

- `ssot-contracts==X.Y.Z`
- `ssot-views==X.Y.Z`
- `ssot-codegen==X.Y.Z`
- `ssot-registry==X.Y.Z`
- `ssot-cli==X.Y.Z`
- `ssot-tui==X.Y.Z`
